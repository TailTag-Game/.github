# TailTag

TailTag is a social convention game where attendees discover and catch participating fursuit characters.

The project is currently being rebuilt from the ground up by a community of contributors.

## Project status

The rebuild is in its foundational planning and setup phase. The initial team is establishing the product direction, architecture, development workflow, and contributor processes.

## Repositories

- [`tailtag`](https://github.com/TailTag-Game/tailtag) — Main TailTag application and active development
