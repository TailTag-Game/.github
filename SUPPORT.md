# TailTag Support

TailTag is currently under active redevelopment and is not yet available as a supported public release.

## Contributor help

Accepted contributors should use the appropriate TailTag Discord help or squad channel for onboarding and development questions.

## Bugs and feature requests

Once public testing begins, bug reports and feature requests will be submitted through the issue forms in the main TailTag repository.

## Security issues

Do not report security vulnerabilities publicly. Follow the instructions in our [Security Policy](SECURITY.MD).
